# belajar-opencv
Mendeteksi wajah dan mata menggunakan library open-cv python

```
$ pip install -r requirements.txt
```

Mendeteksi Wajah dan Mata secara Realtime dengan Webcam
```
$ python app.py
```

Mendeteksi Wajah dan Mata menggunakan path image
```
$ python app-image.py
```
